## Install
- git clone https://github.com/iAmRivard/blog-laravel.git
- cd blog-laravel
- composer install
- create database "blog_laravel" and configure .env file
- php artisan make:refresh --seed
- php artisan serve
- see: http://127.0.0.1:8000/
